# Project Statement: AI Laptop Price Predictor

## 1. Project Title
**AI-Driven Market Valuation: A Regression-Based Laptop Price Predictor**

## 2. Problem Definition
In the current electronics market, the pricing of laptops is influenced by a complex interplay of hardware specifications. Consumers and resellers often lack a standardized tool to determine the "fair market value" of a device based on its core components. 

This project seeks to solve this by creating an **Intelligent Agent** that uses **Supervised Learning** to analyze the relationship between:
* **RAM (GB)**: Primary memory capacity.
* **CPU Speed (GHz)**: Processing power frequency.
* **SSD Capacity (GB)**: High-speed storage volume.

The goal is to provide a real-time price estimation and categorize the device into market segments (Budget, Mid-range, or High-end).

## 3. Objectives & Scope
Consistent with the **CSA2001** syllabus, the objectives are:
* **Mathematical Modeling**: Implementing **Linear Regression** to find the optimal weights for hardware features.
* **Feature Importance**: Identifying which hardware component has the highest coefficient (impact) on the final cost.
* **Usability**: Providing a "Safe Input" interface to allow non-technical users to interact with the ML model.
* **Model Validation**: Using **Mean Absolute Error (MAE)** and **R2 Score** to ensure the predictor’s reliability.

## 4. Relevance to Course Outcomes (COs)
This project serves as a practical application of the following course pillars:
* **CO4**: Analyzing and designing problems using AI techniques.
* **CO5**: Applying machine learning algorithms to solve real-world engineering problems.
* **Statistical Decision Theory**: Utilizing regression outputs to make informed categorical decisions about product tiers.

## 5. Target Audience
* **Students**: For academic research into hardware-price correlations.
* **Consumers**: To verify if a retail price is justified by the internal specifications.
* **Retailers**: To set competitive prices based on data-driven insights.
